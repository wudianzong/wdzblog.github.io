IEDA常用快捷键

1.1 Ctrl + …

常用的快捷键如下：

快捷键 功能
Ctrl + Y 删除当前行
Ctrl + D 复制当前行，直接在下一行复制出当前行
Ctrl + W 选中文本，连续按可以扩大选中范围（以单词为单位）
Ctrl + Enter 在当前行上面插入一行空行。注意：需要在行首使用该快捷键
Ctrl + Backspace 删除单词，可以一下字删除整个单词。注意：需要再单词末尾
Ctrl + / 添加注释和取消注释，【第一次是添加注释，第二次是取消注释】
CTRL + E 查找最近打开的文件
Ctrl + O（Override 重写) 快速重写基类或者接口中的方法
Ctrl + 空格 提示代码，在输入代码时可以提示关键字
Ctrl + H 显示类结构图（类的继承层次）
Ctrl + G 快速定位到指定行指定列
Ctrl + B 快速定位源码，将光标放在方法上输入Ctrl + B , 可以去到该方法的源码
Ctrl + U 快速转到当前类的父类

1.2 Alt + …

常用的快捷键如下：

快捷键 功能
Alt + / 快速补全代码
Alt + Enter 快速提示完成，在代码存在语法问题时，IDEA 会提示使用该快捷键自动修正语法
（功能非常强大，最常使用）
Alt + Insert 快速生成含有任意形参的构造器和私有属性的Getter/Setter等方法（最常使用

1.3 Shift + …

常用的快捷键如下：

快捷键 功能
Shift + Enter 在当前行下面生成一行空行，光标自动移到空行行首
F2 或 Shift + F2 快速定位高亮错误或警告位置
Shift + F6 批量修改变量名，先选中变量名，然后点击快捷键

1.4 Ctrl + Alt + …

常用的快捷键如下：

快捷键 功能
Ctrl + Alt + L 快速格式化代码
Ctrl + Alt + I 自动缩进行
Ctrl + Alt + T 将选中的代码使用 if、while、try/catch 等代码块包装起来（功能强大）
Ctr + Alt + B 查找一个类/ 接口的实现类，如果只有唯一的实现类则可以直接进入该实现类
